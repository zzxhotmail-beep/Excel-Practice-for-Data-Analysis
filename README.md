# Excel & Google Sheets Data Analysis Practice
This repository documents my self-learning journey in spreadsheet-based data analysis, focusing on practical skills application and end-to-end project delivery.

## Repository Introduction
I learned core data analysis skills through Excel-focused tutorials (from YouTube) and completed all hands-on practices in Google Sheets. This repository showcases my proficiency in both tools, covering the entire data analysis workflow-from data cleaning to interactive dashboard building.

## Core Skills & Practices
- Pivot Tables (including Calculated Fields application)
- Essential Spreadsheet Functions (with XLOOKUP in-depth practice)
- Conditional Formatting & Data Formatting
- Data Cleaning & Preprocessing Techniques
- Data Visualization with Charts
- Interactive Slicers for multi-dimensional filtering

## Key Project: Bike Sales Dashboard
A complete end-to-end data analysis project using bike sales datasets, integrating all core skills mentioned above:
- Performed thorough data cleaning to ensure data accuracy
- Applied advanced functions and Pivot Tables for data aggregation
- Analyzed the impact of customer commute distance and age brackets on bike purchase behavior
- Built interactive charts and integrated Slicers for flexible multi-dimensional filtering
- Designed a comprehensive, user-friendly Bike Sales Dashboard to visualize key insights
